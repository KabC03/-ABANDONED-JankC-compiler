## Folder structure overview

- data

    Contains data files

    - Source file (source.txt)
    - Compiler output (IR_source.txt)
    - Assembler output (asm_out.txt)


- documentation

    Contains documentation and TODO


- output

    Compiled .c files



- run

    bash scripts used to compile and run .c files


- src

    source .c files



- unused

    unused .c files
